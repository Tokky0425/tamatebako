# `@smarthr/create-lint-set`

各種リントツールを一括でインストールするための CLI です。

## Usage

```bash
$ npx @smarthr/create-lint-set
```

## Options

第1引数に対象となるディレクトリ名を指定できます。
デフォルト値は `.` です。

```bash
$ npx @smarthr/create-lint-set your/project/directory
```
